⚙️ Codex CLI 前置說明與使用指南

本 Dashboard 主要是解析並呈現 Codex CLI 的本地會話事件與 Token 數據。因為 Codex CLI 會自動將會話記錄在 ~/.codex/sessions 目錄下，您不需要進行任何腳本設定或狀態列修改，本 Dashboard 就會自動讀取並進行即時同步！

1️⃣ 1. 確認 Codex CLI 已在使用

只要您在終端機中執行過 Codex CLI 進行過對話，會話檔案就會自動生成在此目錄中。

2️⃣ 2. 偵測路徑狀態

目前偵測到的 Codex 目錄為：/home/chenting/.codex。本系統會以極高效率對該目錄進行即時掃描與分析。

3️⃣ 3. 費用估算邏輯

費用會根據您的對話模型以 Token 數量為基準進行即時估算，排除快取部分後得出實際成本。
